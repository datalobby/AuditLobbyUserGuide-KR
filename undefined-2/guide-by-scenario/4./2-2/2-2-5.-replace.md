# 2-2-5. 파일 교체하기 \(Replace\)

## 1. 감사조서를 우클릭하여 교체\(Replace\)를 선택합니다. 

![](../../../../.gitbook/assets/file_replace_1.jpg)

## 2. 교체할 파일을 선택합니다. 

{% hint style="info" %}
현재는 원래의 감사조서와 동일한 양식의 파일로만 교체할 수 있습니다. 

차기 업데이트에서 다양한 양식의 파일과도 교체가 가능하도록 개선될 예정입니다. 
{% endhint %}
