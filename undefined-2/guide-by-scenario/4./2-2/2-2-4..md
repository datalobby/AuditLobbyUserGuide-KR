# \(ENG\)2-2-4. Uploading multiple files

## 1. Project View &gt; Workpaper File Menu &gt; Create File  

1. Select 'Workpaper File' from the left menu list of the project screen
2. Click 'Create File' button on the left side of the screen
3. Select 'Multiple Files' on the top of the 'Create File' window'

![&apos;Workpaper File&apos; &amp;gt; &apos;Workpaper File&apos; &amp;gt; &apos;Create File&apos;](../../../../.gitbook/assets/2-2-4-_multiple_file_upload_1.jpg)

![&apos;Multiple Files&apos; is selected in the &apos;Create File&apos; screen. ](../../../../.gitbook/assets/2-2-4-_multiple_file_upload_2.jpg)

## 2. Select Multiple  Files

1. Click 'Upload Files' button on the right of the 'Create File' window to select multiple files

![](../../../../.gitbook/assets/2-2-4-_multiple_file_upload_3-1.jpg)

## 3. Use 'Separating Character' to separate 'Ref Num.' from origin workpaper title

![&apos;Ref Num.&apos; is separated from origin name of file based on underbar\(&apos;\_&apos;\).   ](../../../../.gitbook/assets/2-2-4-_multiple_file_upload_5.jpg)

## 4. Click 'Create' button' 

1. Click 'Create' button and wait a moment
2. The progress of the upload is displayed next to each file

![](../../../../.gitbook/assets/2-2-4-_multiple_file_upload_4.jpg)

![](../../../../.gitbook/assets/2-2-4-_multiple_file_upload_6.jpg)
